# My Portfolio Website 🚀

Welcome to my personal portfolio! This project showcases my skills as a Junior Full Stack Web Developer, featuring modern web animations and a responsive design.

## 🔗 Live Demo
Check out the live website here: [https://han-lin-aung.github.io/MyPortfolio-Website/](https://han-lin-aung.github.io/MyPortfolio-Website/)

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Animations:** GSAP (GreenSock Animation Platform)
* **Icons:** Remix Icons / Font Awesome

## ✨ Key Features
* **Smooth Animations:** Integrated GSAP for high-quality web interactions.
* **Responsive Design:** Optimized for Mobile, Tablet, and Desktop views.
* **Modern UI:** Clean and professional interface focused on user experience.

## 📂 Project Files
* `index.html` - The main structure of the site.
* `style.css` - Custom styling and layout.
* `portfolio.js` - GSAP animation logic and interactivity.
